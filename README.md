# D&G Collection

This is a web site for a school where you can apply get in touch or may be contact the school for more information about the school.

## Features

1 Application Page
2 Contact-Us
3 FAQS Page
4 Programs Page
5 Blog Page

## Technologies Used

Front-end: Next.js, Tailwind
Back-end: CMS-SANITY
